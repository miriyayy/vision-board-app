# Vision Board Generator App

An AI-assisted mobile app that generates natural, non-symmetrical vision boards
based on user-selected keywords and screen ratios.

## Features
- Keyword-based image search (Unsplash API)
- Automatic collage generation
- Phone & desktop screen ratios
- Regenerate layouts
- Expo + React Native

## Tech Stack
- Expo (React Native)
- Expo Router
- Unsplash API

## Status
MVP completed. Collage density and export features in progress.
