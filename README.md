# Vision Board Generator

An AI-assisted mobile application that automatically generates
Pinterest-style vision boards based on user-defined keywords and
target screen ratios.

The app focuses on producing **aesthetic, fully-filled collages**
with both organic and symmetric layout options, optimized for
phone and desktop wallpapers.
